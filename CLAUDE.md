# CLAUDE.md — Modelo Predictivo de Riesgo Pay

## Descripción

Proyecto de modelo predictivo de riesgo para el producto Pay de Superia Capital. Su objetivo es analizar y predecir el riesgo crediticio asociado a las operaciones del producto Pay mediante técnicas de modelado estadístico y machine learning.

---

## Estructura del proyecto

| Carpeta | Contenido |
|--------|-----------|
| `data/` | Bases de datos crudas y procesadas utilizadas para el modelo |
| `codigo/` | Scripts y notebooks con el desarrollo del modelo predictivo |
| `notas/` | Notas de trabajo, observaciones y decisiones del proyecto |
| `reporte/` | Reportes con resultados, métricas y conclusiones del modelo |
| `docs/` | Documentación del proyecto, procesos y referencias |

---

## Convenciones

- El contenido de los archivos debe estar escrito en **español**.
- Los nombres de archivos y carpetas deben estar en **inglés** (ej. `model-results.md`, `data-cleaning.py`).

---

## Reglas

- **Nunca modificar archivos dentro de `data/` sin permiso explícito del usuario.**
- Siempre crear una nueva **branch** para cambios grandes o que afecten múltiples archivos.

---

## Preferencias

- Responder siempre en **español**.
- Usar **tablas** para presentar datos comparativos.
- Incluir un **resumen ejecutivo** al inicio de cualquier reporte generado.
