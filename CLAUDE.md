# Instrucciones para Claude Code - Proyecto de Practica Superia Capital

## Descripcion del proyecto

Este es un proyecto de practica creado durante el programa de AI Enablement de Superia Capital. Su proposito es aprender a usar Claude Code para gestionar proyectos. No es un proyecto de produccion.

## Estructura de carpetas

```
PROYECTOS/
├── docs/        # Documentacion del proyecto
├── data/        # Archivos de datos de entrada
├── reports/     # Reportes generados
├── .gitignore   # Exclusiones de Git
├── CLAUDE.md    # Este archivo
└── README.md    # Descripcion general del proyecto
```

## Convenciones

- **Idioma del contenido:** todos los archivos se escriben en español
- **Nombres de archivo:** usar ingles y formato kebab-case (ej. `financial-report.md`, `notas-del-programa.md`)

## Reglas importantes

- **No modificar archivos dentro de `data/`** sin permiso explicito del usuario. Siempre pedir confirmacion antes de crear, editar o eliminar cualquier archivo en esa carpeta.

## Preferencias de respuesta

- Responder siempre en **español**
- Usar formato **Markdown** en las respuestas
- Ser conciso y directo
